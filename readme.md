# Next Int

Live version: [https://thinkific.akiosakae.com/](https://thinkific.akiosakae.com/)

## How to run
```javascript
git clone git@github.com:akiokio/NextInt.git
cd NextInt
yarn install
mv sample.env .env
yarn start
```

The project will be availabe at: http://localhost:3000
